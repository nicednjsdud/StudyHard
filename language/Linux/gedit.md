      - 리눅스에서 주로 사용되는 vim(vi) 에디터가 불편하면, gedit라는 간단한 편집기를 대신사용 할수 있음
      - vim 과달리 전문적인 기능은 x , 쉽고 간편함에 초점을 맞춘 편집기
      - 그놈(GNOME) 데스크탑용인데, 우분투 DeskTop CD로 리눅스를 설치하면 기본적으로 설치
