3. GitHub
    1) Git 기반
    2) 프로젝트 호스팅 사이트
    3) 2008년 2월 서비스 시작

    Bash 명령어 
    1) git config --list (목록)
    2) git config --global user.name || user.email 이름이나 이메일 지정
    3) pwd 위치!
    4) cd.. 상위
    5) ls 리스트 
    6) clear 깨끗하게하기
    7) vim HelloWorld!.java --> java파일 하나생김
        - 나갈때는 insert-> i :wq 누르면 저장됨

    8) ls -a 상위폴더 안나타나고 현재 파일만 보임 
    9) ls -al 시간까지

    10) cat Helloworld.java 
            --> 파일 내용 보여줌
