Swagger
=======

## 1. 스웨거(Swagger)란?

* Web API 문서화를 위한 도구
* API들이 가지는 명세(Spec)을 관리하기 위한 프로젝트
* Web API를 수동으로 문서화 하는 것은 굉장히 힘든 작업이다.
    * Swagger를 사용하면 문서가 자동으로 갱신

## 2. 스웨거의 기능

* API Design
* API Development
* API Documentation
* API Testing
* API Mocking and Virtualization
* API Governance
* API Monitoring
* OpenAPI & Swagger
